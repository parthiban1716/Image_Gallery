🖼 Image Gallery Web Application
**********************************
The Image Gallery is a responsive web application that displays a collection of images in an organized and visually appealing layout. It allows users to browse images smoothly while demonstrating modern front-end development techniques.

This project highlights the use of structured layouts, responsive design, and clean UI styling to create an engaging gallery experience across different screen sizes.

🚀 Features
*****************
🖼 Organized image gallery layout
📱 Fully responsive design (mobile, tablet, desktop)
🎨 Clean and modern user interface
⚡ Smooth browsing experience
📂 Structured and maintainable project files

🛠 Tech Stack
****************
HTML5

CSS3

Responsive Web Design

Flexbox / CSS Grid
